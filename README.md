# Art-Spark

## About 

Art-Spark is a tool for artists, designed to give them artistic punch when they run out of ideas. Art-spark uses combinations of cards within categories: 
- Composition
- Color
- Pattern
- Palette
- Shapes
- Symbol
- Object
- Constraints
- Texture
- Shades

## How to use
Select desired decks by clicking on them.

<img width="1920" height="900" alt="image" src="https://github.com/user-attachments/assets/34dea78d-6e1c-4a68-8283-7a0c08bff0d0" />

After selecting the desired decks click <mark style="background-color: lightblue">➜</mark> to generate the cards.

<img width="1920" height="901" alt="image" src="https://github.com/user-attachments/assets/5cd40725-4cb3-4c30-9c79-2e6d6a37a8c9" />



The program randomly pulls cards from the decks, selected by user and displays them. User can regenerate any card by clicking on it, or regenerate all cards on the screen by clicking <mark style="background-color: lightblue">↻</mark>. Also they can go back and select another decks.

<img width="1920" height="900" alt="image" src="https://github.com/user-attachments/assets/71c0395c-15ad-4986-ab0f-ba3dbf1becc0" />
<img width="1920" height="904" alt="image" src="https://github.com/user-attachments/assets/8b495e42-9534-406e-a1c7-bf1959052148" />
